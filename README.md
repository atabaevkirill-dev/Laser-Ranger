# Система лазерного дальномера с видео-оверлеем

## Обзор проекта

Laser Ranger — это программное обеспечение для десктопного приложения, предназначенное для интеграции лазерного дальномера и сетевой камеры. Проект предоставляет решение для получения данных о расстоянии и наложения графических элементов на видеоизображение в реальном времени, что особенно полезно для задач промышленного контроля, интеллектуальной безопасности или навигации роботов.

## Основные функции

### 1. Интеграция лазерного дальномера
- Поддержка подключения дальномера через последовательный порт или TCP/IP
- Возможность одиночных и непрерывных измерений
- Поддержка различных частот измерений (1-10 Гц)

### 2. Поддержка IP-камер
- Подключение к камерам через RTSP-поток
- Совместимость с ONVIF-устройствами
- Возможность управления поворотом и масштабированием камеры (PTZ)

### 3. Графический оверлей
- Наложение военных прицелов на видеоизображение
- Поддержка 5 типов целей: пехота, транспорт, самолет, здание, БПЛА
- 5 стилей прицела: перекрестие, милдот, BDC, круг, тактический
- 6 цветов оверлея для лучшей видимости
- Отображение информации на экране (HUD)
- Шкала расстояний

### 4. Расширенные возможности настройки
- Регулировка масштаба, прозрачности, яркости оверлея
- Режим градаций серого для лучшей видимости при определенных условиях освещения
- Настройка толщины линий прицела
- Поддержка черно-белого режима изображения

## Архитектура проекта

### Основные компоненты

#### [main.py]
- Главное окно приложения с интерфейсом пользователя
- Управление подключениями и потоками
- Визуализация камеры и оверлея

#### [overlay_renderer.py]
- Рендеринг графических элементов поверх видео
- Реализация различных стилей прицелов и силуэтов целей
- Поддержка настроек оверлея (толщина линий, градации серого и др.)

#### [onvif_camera_control.py]
- Классы управления ONVIF-совместимыми камерами
- Поддержка протокола Pelco-D для управления поворотом камеры

#### [protocol_handler.py]
- Обработка протоколов связи с дальномером
- Поддержка различных форматов данных

#### [tcp_protocol_handler.py]
- Обработка TCP/IP-соединений с дальномером

#### [config.ini]
- Файл конфигурации с настройками подключения и параметрами системы

## Технологии

- **Графический интерфейс**: PyQt5
- **Обработка видео**: OpenCV (opencv-python-headless)
- **Вычисления**: NumPy
- **Связь по последовательному порту**: PySerial
- **ONVIF-протокол**: onvif_zeep
- **Конфигурация**: стандартный модуль configparser

## Особенности реализации

### Режим градаций серого
- Добавлена возможность переключения видеоизображения в черно-белый режим
- Улучшает видимость объектов при определенных условиях освещения
- Реализовано через преобразование цветового пространства BGR в GRAY и обратно

### Регулировка толщины линий
- Пользователь может изменять толщину линий прицела с помощью слайдера
- Диапазон настройки от 1.0× до 5.0×
- Толщина линий учитывает масштаб прицела для согласованного отображения

## Установка и запуск

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Настройте [config.ini](file:///C:/Users/White_/Desktop/Laser%20Ranger/config.ini) с вашими параметрами подключения

3. Запустите приложение:
   ```
   python main.py
   ```
