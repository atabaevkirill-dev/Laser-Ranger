# Новые возможности системы лазерного дальномера

## Режим градаций серого

### Описание
Добавлена возможность переключения видеоизображения в черно-белый режим, что улучшает видимость объектов при определенных условиях освещения.

### Техническая реализация
- Добавлен параметр [gray_mode](file:///C:/Users/White_/Desktop/Laser%20Ranger/main.py#L408-L408) в класс [OverlayState](file:///C:/Users/White_/Desktop/Laser%20Ranger/overlay_renderer.py#L275-L291)
- В функции [render](file:///C:/Users/White_/Desktop/Laser%20Ranger/overlay_renderer.py#L293-L348) добавлена проверка этого параметра
- При включении режима изображение преобразуется из цветного (BGR) в градации серого и обратно в BGR для совместимости с остальным кодом
- В интерфейсе добавлен чекбокс "Grayscale mode" в группу управления оверлеем

### Использование
1. Запустите приложение
2. В группе "Target Overlay Controls" найдите чекбокс "Grayscale mode"
3. Установите или снимите галочку для включения/выключения режима

### Преимущества
- Улучшенная контрастность изображения в условиях низкой освещенности
- Лучшая видимость объектов на определенных фонах
- Помощь в выделении контуров и форм объектов

## Регулировка толщины линий прицела

### Описание
Добавлена возможность изменения толщины линий прицела для лучшей видимости и адаптации к различным условиям наблюдения.

### Техническая реализация
- Добавлен параметр [line_thickness](file:///C:/Users/White_/Desktop/Laser%20Ranger/overlay_renderer.py#L293-L348) в класс [OverlayState](file:///C:/Users/White_/Desktop/Laser%20Ranger/overlay_renderer.py#L275-L291)
- Формула расчета толщины линий изменена с фиксированной на: `lw = max(1, int(st.line_thickness * st.scale))`
- В интерфейсе добавлен слайдер "Line thickness" с диапазоном от 1.0× до 5.0×
- Добавлен метод [_ov_thickness_changed](file:///C:/Users/White_/Desktop/Laser%20Ranger/main.py#L888-L890) для обработки изменений толщины

### Использование
1. Запустите приложение
2. В группе "Target Overlay Controls" найдите слайдер "Line thickness"
3. Перемещайте слайдер для изменения толщины линий прицела
4. Значение отображается рядом с ползунком (например, "1.5×")

### Преимущества
- Возможность сделать прицел более или менее заметным
- Адаптация к различным условиям просмотра
- Улучшенная видимость прицела на разных типах фона

## Совместимость с другими функциями

Обе новые функции полностью совместимы с существующими возможностями:
- Работают вместе с регулировкой масштаба, прозрачности и яркости
- Совместимы с различными стилями прицелов и силуэтами целей
- Не влияют на производительность системы
- Интегрированы в общую систему настроек оверлея

## Примеры использования

### В условиях низкой освещенности
- Включите режим градаций серого для улучшения контрастности
- Увеличьте толщину линий прицела для лучшей видимости

### При работе с мелкими деталями
- Уменьшите толщину линий прицела, чтобы не перекрывать важные элементы
- Используйте цвета с высокой контрастностью

### При длительной работе
- Настройте параметры под свои предпочтения для снижения усталости глаз
- Используйте толщину линий, обеспечивающую комфортную визуализацию
