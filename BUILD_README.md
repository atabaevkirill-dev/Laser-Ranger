# Сборка исполняемого файла Laser Ranger

Этот документ описывает процесс создания исполняемого файла (.exe) для приложения Laser Ranger.

## Подготовка

Прежде чем начать сборку, убедитесь, что у вас установлен Python 3.8 или выше, а также все зависимости проекта.

## Автоматическая сборка

Для автоматической сборки исполняемого файла выполните следующую команду в корневой директории проекта:

```bash
python build_exe.py
```

Этот скрипт:
1. Установит все зависимости из `requirements.txt`
2. Установит PyInstaller если он не установлен
3. Соберет исполняемый файл с помощью PyInstaller

## Ручная сборка

Если вы предпочитаете собирать вручную, выполните следующие шаги:

### 1. Установите зависимости:
```bash
pip install -r requirements.txt
pip install pyinstaller
```

### 2. Выполните сборку с помощью PyInstaller:
```bash
pyinstaller laser_rangefinder.spec
```

Или используйте команду:
```bash
pyinstaller --onefile --windowed --name=LaserRangefinder --icon=icon.ico main.py
```

## Результаты сборки

После успешной сборки исполняемый файл будет расположен в папке `dist/LaserRangefinder.exe`.

## Замечания

- Приложение имеет графический интерфейс, поэтому сборка выполняется с флагом `--windowed`, чтобы не открывалось консольное окно.
- Файл конфигурации `config.ini` будет включен в исполняемый файл.
- Все зависимости, такие как PyQt5, OpenCV и другие, будут включены в итоговый исполняемый файл.
- Если у вас есть файл иконки `icon.ico`, он будет использован для оформления исполняемого файла.

## Возможные проблемы

1. Если сборка завершается с ошибками, возможно, потребуется установить дополнительные зависимости или изменить файл спецификации.
2. Размер итогового исполняемого файла может быть значительным из-за включения всех зависимостей.
3. Первый запуск собранного приложения может занимать больше времени, так как происходит распаковка зависимостей.
