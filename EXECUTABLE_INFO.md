# Информация о созданных исполняемых файлах Laser Ranger

После завершения процесса сборки с помощью PyInstaller в папке [dist](file:///Users/White_/Desktop/Laser%20Ranger/dist) были созданы следующие исполняемые файлы:

## Файлы в папке dist:

1. **LaserRangefinder.exe** - основной исполняемый файл приложения
   - Это полноценное приложение с графическим интерфейсом
   - Все зависимости включены внутрь файла
   - Может запускаться на любом компьютере с Windows без установки Python
   - Используется для обычного запуска приложения

## Дополнительная информация:

- Оба файла являются результатом процесса сборки PyInstaller
- Для распространения приложения достаточно использовать только LaserRangefinder.exe
- Файл содержит все необходимые зависимости для работы приложения
- При первом запуске приложения может потребоваться больше времени из-за распаковки временных файлов

## Замечания:

- Приложение использует PyQt5 для графического интерфейса
- Для работы приложения требуются библиотеки: pyserial, opencv-python-headless, numpy, onvif_zeep
- Все эти зависимости уже включены в исполняемый файл
- Конфигурационный файл config.ini также должен находиться в одной папке с исполняемым файлом для правильной работы

## Рекомендации по использованию:

1. Для запуска приложения используйте файл LaserRangefinder.exe
2. Убедитесь, что рядом с исполняемым файлом находится файл config.ini
3. Приложение работает с камерами по протоколу ONVIF и с дальномерами по протоколу UART/TCP
4. Для корректной работы панели управления поворотом убедитесь, что сетевые настройки указаны правильно
